# Веб-приложение Мессенджер

Учебный проет в рамках курса "Мидл-фронтенд разработчик" от Яндекс Практикума  

---

### Установка зависимостей:

#### `npm install`

### Запуск приложения:

#### `npm run dev`

### Cборка (build) и запуск express-сервера на <a>http://localhost:3000/</a>:

#### `npm run start`
